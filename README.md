## 🚀 Quick Start

### 30 Seconds to First Scan

```bash
# Basic scan
python3 modbus_scanner.py -t 192.168.1.100

# With report
python3 modbus_scanner.py -t 192.168.1.100 -j report.json

# Verbose mode
python3 modbus_scanner.py -t 192.168.1.100 -v
```

---

## 📋 Usage

```bash
# Basic
python3 modbus_scanner.py -t <TARGET_IP>

# Comprehensive
python3 modbus_scanner.py -t 192.168.1.100 -j report.json -v

# Custom port
python3 modbus_scanner.py -t 192.168.1.100 -p 20502

# Help
python3 modbus_scanner.py -h
```
